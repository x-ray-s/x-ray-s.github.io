# Windows 设置窗口置顶

下载 AutoHotkey (opens new window)

右键创建 ahk 脚本.

编辑器打开输入

^SPACE::  Winset, Alwaysontop, , A

使用 ctrl + space 来设置窗口最上层。

运行脚本。

在要设置指定的应用中，使用快捷键 设置/取消 置顶功能。
